# To run tests with JEST

   ```bash
   cd EcoPaths/frontend
   npm run test:jest
   ```

   ## Testing with ROBOT

### Install robot framework

```bash
npm run install-robot
```

### Running tests

1. **Start the app on terminal 1**
```bash
npm start
```

2. **Run tests on terminal 2**
```bash
npm run test:robot
```
or
```bash
npm run test:robot:headless
```
